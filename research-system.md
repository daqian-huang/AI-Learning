# 我的 AI 投资研究系统

这个仓库用于记录我如何使用 GitHub、Codex 和 AI，提高 Bitcoin、黄金和美股研究的效率。

## 我的目标

我不是为了成为职业程序员，而是为了建立一个实用的研究工作流：

- 用 GitHub 保存和整理研究资料
- 用 Markdown 写清楚自己的市场观察
- 用 AI 总结新闻、研报和财报内容
- 用 Codex 帮我创建简单工具和整理文件
- 每周复盘自己的判断，减少重复劳动

## 我主要研究的资产

| 研究方向 | 我关注什么 |
|---|---|
| Bitcoin | 价格趋势、ETF 资金流、链上数据、宏观流动性 |
| 黄金 | 美元指数、美债收益率、实际利率、通胀和避险需求 |
| 美股 | 公司基本面、财报、估值、行业趋势和市场情绪 |

## 我的基本工作流

### 1. 收集信息

每天或每周收集重要信息，例如：

- Bitcoin 相关新闻
- 黄金和宏观数据
- 美股公司财报和市场新闻
- 美联储、通胀、利率和美元相关信息

### 2. 用 AI 做初步整理

我会让 AI 帮我：

- 总结长文章
- 提取重点
- 区分事实和观点
- 找出潜在风险
- 生成周报草稿

### 3. 人工判断

AI 不能替我做投资判断。我需要亲自检查：

- 数据是否真实
- 来源是否可靠
- 结论是否过度自信
- 是否遗漏了反方观点
- 是否需要继续验证

### 4. 保存到 GitHub

重要内容会保存到这个仓库中，方便以后复盘。

建议分类：

- `bitcoin/`：Bitcoin 研究笔记
- `gold/`：黄金研究笔记
- `stocks/`：美股研究笔记
- `prompts/`：常用 AI 提问模板
- `templates/`：研究报告模板
- `reports/`：周报、月报和复盘
- `tools/`：以后放简单自动化工具

## 每周固定动作

每周至少完成一次：

1. 更新 Bitcoin、黄金和美股的观察
2. 总结本周最重要的市场变化
3. 写下自己的判断
4. 列出下周需要关注的问题
5. 记录自己可能错在哪里

## 使用 AI 的原则

> AI 负责整理，我负责判断。

我可以让 AI 帮我提高效率，但不能完全相信 AI 的结论。尤其是金融市场相关信息，必须自己验证数据和来源。
